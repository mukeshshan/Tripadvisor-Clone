TripAdvisor Clone Website

This project is a TripAdvisor Clone Website built using HTML and CSS. The website design is inspired by the popular travel platform Tripadvisor.

🚀 Features

Responsive homepage layout

Navigation bar with search option

Travel category sections

Hotel and destination cards

Hover effects and modern UI design

🛠️ Technologies Used

HTML5

CSS3

Flexbox

Responsive Design

📄 Pages

Home Page – Main landing page with travel sections

🎯 Purpose of the Project

The main purpose of this project is to practice frontend web development and improve skills in HTML layout, CSS styling, and responsive design.
